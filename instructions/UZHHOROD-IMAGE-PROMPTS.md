## Style Guidelines for All Images

### Color Palette:
- Primary: Blue tones (matching the website's blue-600 theme)
- Secondary: Warm whites and grays
- Accent: Natural greens (plants)
- Avoid: Overly saturated or dark images

### Mood & Atmosphere:
- **Professional** but approachable
- **Bright** and well-lit
- **Modern** and contemporary
- **Trustworthy** and welcoming
- **Warm** and inviting

### Technical Requirements:
- High resolution (minimum 1200px on shortest side)
- Professional quality photography
- Clean, modern aesthetic
- Appropriate aspect ratios as specified
- Web-optimized (consider generating in WebP format if possible)

### Post-Processing Suggestions:
- Slight color grading to match website blue theme
- Ensure proper contrast and brightness
- Optimize file size for web (80-85% quality)
- Add subtle gradient overlays if needed (especially for city-view.jpg)

## Notes for Image Generators

### For DALL-E 3:
Use the full prompts as provided above. DALL-E 3 understands aspect ratios well.

### For Midjourney:
Add `--ar 4:3` or `--ar 21:9` or `--ar 16:9` at the end of prompts.  
Consider adding `--style raw` for more realistic results.

### For Stable Diffusion:
Use negative prompts like:
- "dark, gloomy, low quality, blurry, distorted, cartoon, illustration"
- Add positive prompts: "professional photography, high quality, realistic, sharp focus"

### For Stock Photo Sites:
Use the provided keywords. Search terms are optimized for Unsplash, Pexels, and Pixabay.
